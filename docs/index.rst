AFIM Python Package Documentation
=================================

AFIM (Antarctic Fast Ice and Pack Ice Metrics) is a Python library to analyse and visualise CICE model output, focused on sea ice processes.

Installation
------------

Clone the GitHub repository and install:

.. code-block:: bash

    git clone https://github.com/dpath2o/AFIM.git
    cd AFIM
    pip install -e ./src

Or use the Conda environment file:

.. code-block:: bash

    conda env create -f src/environment.yml
    conda activate afim
